Project: Drawing Tool

Description

The main purpose of the project was to build an interactive application - Drawing Tool. The tool demonstrates programming with HTML Canvas, JavaScript and jQuery (mouse events).

Developing Tools

  • HTML Canvas
  • CSS3
  • JavaScript
  • jQuery

Features

  • The tool allows the user to draw on the canvas with a mouse.
  • The user can pick a color from the color picker and change the line width with a slider.
  • The erase button allows to erase any spot on the drawing.
  • The user can reset the activity at any time and start drawing on an empty canvas.
  • The user can save the drawing and access it after opening the page again.
