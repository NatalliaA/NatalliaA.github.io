Project: Online Lesson and Quiz

Description

The main purpose of the project was to build a responsive webpage - Online Lesson and Quiz. The webpage displays different sections of the online lessons (photo gallery, text, quiz, video and research assignment) and demostrates the use of Bootsrap for responsive styling and jQuery for creating the quiz.

Developing Tools

  • HTML5
  • CSS3
  • Bootstrap
  • jQuery

Features

  • The webpage is styled with Bootstrap and includes responsive elements such as jumbotron, cards, video, carousel with the photo gallery, and etc.
  • The Quiz section adds interactivity for the webpage and is created with jQuery. The quiz has the following elements:
    • A country is displayed in a random order (1 out of 12).
    • There are four multiple answer choices with only one correct answer.
    • If the correct (incorrect) answer is selected, the "right" label ("wrong" label) is shown for one second.
    • 12 displayed maps will be colored one at a time: green for the right answer and red for the wrong answer.
    • The score window updates the number of correct answers.
    • The player can reset the game and start a new quiz at any time.
    • The game includes music sounds (wrong, correct, game over) for creating a playful mood.
    • After 12 questions, the game over window is displayed with the total score number and the achievement sentence.
  • The links to external websites provided in the Research section make it convinient for the student to find information for the assignment.
